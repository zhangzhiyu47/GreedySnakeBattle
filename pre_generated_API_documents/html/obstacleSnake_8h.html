void obsEatWallsOrUserSnake ( GameAllRunningData * data)

Judge whether the obstacle snake eats the wall or the user snake.

Judge whether the obstacle snake eats the wall or the user snake. If obstacle snake eat them,it will change the status(GameAllRunningData.obsState) of the obstacle snake(to 1 or 2, see GameAllRunningData.obsState for specific meaning) to describe the situation of obstacle.
