◆ disableNormalInput()

void disableNormalInput ( const pid_t childProcess)

Provide an interface for the parent process to let the child process call the disable_normal_input function.

Due to replacing the standard input of the parent process with the read end of the pipeline and treating the written content of the pipeline written by the child process as the standard input, it no longer has the ability to directly set the standard input. This function is provided to the parent process that sends the SIGUSR1 signal to the child process childProcess, causing the child process to call the disable_normal_input function, thereby canceling the standard input buffer and echo.

Parameters
[in]childProcessID of the child process.

◆ enableNormalInput()

void enableNormalInput ( const pid_t childProcess)

Provide an interface for the parent process to let the child process call the enable_normal_input function.

Due to replacing the standard input of the parent process with the read end of the pipeline and treating the written content of the pipeline written by the child process as the standard input, it no longer has the ability to directly set the standard input. This function is provided to the parent process, which sends the SIGUSR2 signal to the child process childProcess, causing the child process to call the enable_norms_input function, thereby enabling the standard input buffer and input echo.

Parameters
[in]childProcessID of the child process.
