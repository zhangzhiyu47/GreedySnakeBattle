Detailed Description

These functions are used for signal capture.

These functions are all used for signals. The naming convention for these functions is:
[father/child]ProcessCatch[signal name (INT/CHLD/USR1/USR2/...)].
For example, the function for the parent process to capture the SIGINT signal is fatherProcessCatchINT.

◆ fatherProcessCatchCHLD()

void fatherProcessCatchCHLD ( int signum)

Father process captures signal SIGCHLD.

Capture the signal SIGCHLD and try to recycle the subprocess to judge whether the subprocess is dead. If dead, exit the game application (that is, the parent process).

Parameters
[in]signumThe number of the captured signal.
