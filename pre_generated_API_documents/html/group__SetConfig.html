Detailed Description

These functions are used for game settings.

These functions are used for game settings. These functions all start with 'setConfig', connected by '_' in the middle, and followed directly by the variable name in the GameConfig to be set (for example, to set GameConfig.isEnableObs, you need to call the setConfig.isEnableObs function). Except for the setConfigAllToDefault function, which is used to restore all settings to default. The parameters of these functions are all GameConfig (i.e. all game configurations).
